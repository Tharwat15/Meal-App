# Meal App

A beautiful, responsive Flutter application designed to browse recipes, view meal categories, and manage dietary preferences. This project was initially bootstrapped using FlutLab and utilizes state management to deliver a seamless user experience.

---

## 🚀 Features

* **Meal Categories:** Browse through a variety of distinct food categories.
* **Detailed Recipes:** View ingredients, step-by-step preparation instructions, and meal details.
* **State Management:** Powered by the `provider` package for efficient and scalable state handling.
* **Custom Typography:** Features beautiful, clean custom font pairings with **Raleway** and **RobotoCondensed**.
* **Cross-Platform:** Built to support multiple platforms including Android, iOS, and Web.

---

## 🛠️ Tech Stack & Dependencies

* **Framework:** [Flutter](https://flutter.dev/) (SDK `>=3.0.0 <4.0.0`)
* **Language:** [Dart](https://dart.dev/)
* **State Management:** [Provider (^6.0.0)](https://pub.dev/packages/provider)
* **Design System:** Material Design

---

## 📦 Project Structure

The project includes standard Flutter configurations, customized assets, and specific tools:
* `lib/main.dart` - The main entry point of the application.
* `pubspec.yaml` - Manages project dependencies, assets, and custom fonts.
* `analysis_options.yaml` - Enforces clean coding standards using `flutter_lints`.
